Harvard Ed Portal - Web Dev Spring '21



Welcome to our Webpage! Six Boston-area high school students took a coding class in the Spring of 2021. They came together to enhance their knowledge on web developement with the final goal being to create a website about the Boston-Area. The finished product is what you see here! After importing all sorts of things into a thing called "CS50.ide", a cloud-based Integrated Development Environment, we were able to start coding right away! The instructors set up challenges for us to get familiar with html. We learned about all the different aspects of a webpage like HTML, CSS and JAVASCRIPT. The class was a lot of fun and we all learned a lot! Feel free to navigate this webpage and learn about the Boston-Area!
